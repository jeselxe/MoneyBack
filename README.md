# MoneyBack
App made with atoms-app to have a list of debts

## Install

```bash
	$ npm install
	$ bower install
	$ gulp init
```

### Start server

```bash
	$ node zen zen
```

It has to be running a MongoDB server to connect to it.

## Run it

```bash
	$ gulp
```

and then <a href="http://localhost:8000">http://localhost:8000</a>
